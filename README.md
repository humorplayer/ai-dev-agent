# AI Dev Agent (Multi-Agent Coding Assistant)

## Overview
A multi-agent AI system designed to automate code analysis, refactoring, and development workflows.

## Core Features
- Repository scanning (AST + semantic analysis)
- Automated refactor plan generation
- Multi-agent collaboration (Planner / Executor / Reviewer)
- Code generation and modification
- Test and lint validation
- Pull request draft generation

## Architecture
- Planner Agent: Task decomposition (long-chain reasoning)
- Executor Agent: Code execution and generation
- Reviewer Agent: Validation and feedback

## Workflow
1. Scan repository
2. Analyze code quality
3. Generate refactor plan
4. Apply code changes
5. Run tests and lint checks
6. Output results

## Environment
- CLI-based execution
- Supports iterative refinement
- Context-aware multi-step reasoning

## Status
Actively used in personal and small-scale team projects.
